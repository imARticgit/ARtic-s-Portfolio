My Projects

Two systems I built to show some of my skills for this portfolio. (Videos are 40-50 seconds long, they will not take much of your time :)

Featured System Combat

Rough Combat System

A quick combat system I made. Features sprinting (LShift), an M1 combo, and a hitbox system. I used UserInputService for both sprinting and M1s. I used ReplicatedStorage to fire remote events to damage the dummy and play animations.

Watch it here!
Featured System PathFinding

Pathfinding Dummy

A pathfinding system using an R6 rig. Features a walking animation and a dash system for the player to escape. I used PathfindingService to track the player's position. It only follows the nearest player within a 200-stud radius.
